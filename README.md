# Blog trip made with chat GTP

Made with DATO CMS and Nuxt 3 (https://www.datocms.com/docs, https://nuxt.com/docs/getting-started/introduction)

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```
